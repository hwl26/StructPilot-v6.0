# StructPilot v5.3 更新日志

**发布日期**: 2026-07-16  
**版本类型**: 高级 UI 优化版

---

## 🎯 核心优化

### 1. 全局字体调整
- 基础字体大小调整为 14px，提升可读性
- 标题层级字体同步调整（h1: 26px, h2: 22px, h3: 18px, h4: 16px）
- 整体视觉更清晰舒适

### 2. 顶部状态工具栏
- 固定在页面顶部的全局状态栏
- 显示当前阶段信息：Step 序号、阶段名称、进度统计
- 右侧显示软件类型（cryoSPARC/RELION）和阶段分类标签
- 一目了然掌握当前工作状态

### 3. 左侧摘要卡片优化
- **2x2 紧凑网格布局**，大幅减少纵向空间占用
  - 左上：🎯 目标 - 显示阶段目标摘要
  - 右上：📥📤 I/O - 显示输入输出要求
  - 左下：📋 步骤 - 显示关键步骤数量
  - 右下：✅ 质控 - 显示质控要点数量
- 点击可展开查看详细内容
- 先看概览，需要时再深入细节

### 4. 工作区布局优化
- **详细工作区设置 600px 固定高度 + 内部滚动**
  - 避免左侧过长挤压右侧对话区
  - SOP/参数/截图/质控 tabs 在容器内独立滚动
- **列宽比例调整为 38:62**
  - 左侧：工作区（更紧凑）
  - 右侧：对话区（更宽敞）
- 信息层级清晰，视觉焦点合理分配

### 5. 独立 QA 验收流程
- **在最新助手回答下方显示快速反馈按钮**
  - 👍 有帮助
  - 👎 不够好
  - ⏭️ 跳过
- **选择"不够好"后展开详细反馈表单**
  - 问题类型选择：不够准确/信息不全/难以理解/其他
  - 具体说明输入框
  - 反馈自动记录到知识审核队列
- **独立于诊断工具**，所有用户都能轻松反馈
- 帮助系统持续改进

---

## 🔧 技术改进

### Bug 修复
- 修复摘要卡片折叠按钮在旧版 Streamlit 中的兼容性问题
- 改用原生 `expander` 组件，更稳定可靠

### 性能优化
- 摘要卡片内容截断显示，减少渲染开销
- 工作区滚动容器隔离，避免全局重排

## 📦 部署说明

### 系统要求
- Python 3.10+
- Windows 10/11 或 macOS 10.15+
- 4GB+ RAM
- 网络连接（用于 LLM API 调用）

### 快速启动
```bash
# 1. 解压 StructPilot_v5.3.zip

# 2. 进入目录
cd StructPilot_v5.3

# 3. 双击运行（Windows）
start.bat

# 4. 浏览器访问
http://localhost:8501
```

### 环境配置
- 编辑 `.env` 文件配置 API Key
- 参考 `.env.example` 查看所有配置选项

## 🎨 UI/UX 亮点

1. **第一视觉中心**：顶部状态栏 → 一眼看到当前位置和进度
2. **第二层级**：左侧摘要卡片 → 快速了解本阶段核心信息
3. **第三层级**：详细工作区 tabs（SOP/参数/截图/质控）→ 深入操作细节
4. **第四层级**：聊天区域 → 交互式问答和验收反馈

信息层级分明，符合用户视觉扫描习惯，从概览到细节逐层深入。
