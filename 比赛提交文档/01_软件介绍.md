# StructPilot v6.0 - 冷冻电镜数据处理智能陪跑助手

## 一、软件简介

**StructPilot** 是一个本地可运行的冷冻电镜（Cryo-EM）单颗粒分析流程智能导航助手。它不替代 cryoSPARC 或 RELION 等专业处理软件，而是作为"陪跑教练"，通过检查点式引导、参数建议、故障排查、多模态输入（图片/语音）和可选 AI 增强，帮助用户顺利完成从原始数据到 3D 结构的全流程处理。

### 核心特点

1. **检查点式流程管理**
   - 将完整流程拆分为 12 个标准检查点（cp_01 数据导入 → cp_12 模型构建与验证）
   - 每个检查点支持"开始/完成/跳过/失败"等状态流转
   - 全程记录进度、参数、质控结果和用户反馈

2. **双软件体系支持**
   - 完整支持 **cryoSPARC** 和 **RELION** 两大主流冷冻电镜处理平台
   - 根据用户选择的软件体系自动适配参数建议和操作指导

3. **多层次回答模式**
   - **简洁模式**：快速给出结论、下一步操作和关键风险
   - **教学模式**：详细解释原理、参数含义、操作步骤和常见错误
   - **专家模式**：提供前提假设、参数边界、QC 标准、替代方案和回退路径

4. **多模态智能交互**
   - **文本问答**：支持流程问题、参数咨询、故障排查、概念解释
   - **图像理解**：上传 CTF、2D class、3D refinement、FSC 等截图，系统自动识别关键信息
   - **语音输入**：口述问题自动转写为文字，适合实验室场景快速提问

5. **渐进式 AI 能力**
   - **基础模式**（无需 API Key）：基于本地规则、SOP、知识库和截图兜底
   - **AI 增强模式**（配置 API 后）：启用语义检索、LLM 改写、图像理解和语音转写

6. **知识库闭环管理**
   - 支持知识文档的导入、审核、分级（正式/草稿/废弃）
   - 追踪知识来源、证据等级、风险等级和使用频次
   - 用户可"沉淀经验"，将有价值的对话内容转化为知识库条目

## 二、应用场景

### 主要用户

1. **冷冻电镜初学者**
   - 快速上手标准处理流程
   - 减少参数设置错误
   - 及时发现和解决常见问题

2. **实验室团队成员**
   - 统一处理规范和质控标准
   - 积累和传承实验室经验
   - 跨会话追踪项目进度

3. **知识工程师/管理员**
   - 维护和更新实验室 SOP
   - 管理故障诊断知识库
   - 审核和评估知识质量

### 典型使用流程

```
1. 选择软件体系（cryoSPARC/RELION）
   ↓
2. 选择回答深度（简洁/教学/专家）
   ↓
3. 按检查点推进流程
   - 汇报"开始处理"
   - 上传截图/语音提问
   - 获取参数建议和质控指导
   - 汇报"完成"或"报错"
   ↓
4. 导出流程报告
   - 完整对话记录
   - 参数使用历史
   - 质控结果汇总
```

## 三、技术亮点

### 1. 架构设计

- **分层解耦**：表现层（Streamlit）/ 编排层（LangGraph）/ 智能体层 / 知识层 / 持久层五层架构
- **单一权威应答**：避免多智能体并发产生重复回复
- **全链路优雅降级**：任一环节（LLM/Embedding/Vision）缺失均不阻断基本功能

### 2. 知识与代码分离

- 工程骨架与知识内容完全解耦
- 空知识库即可运行（基于内置规则）
- 导入实验室经验后能力自动增强

### 3. 持久化与可追溯

- SQLite 数据库存储完整会话、消息、检查点记录和图像元数据
- 支持会话恢复、进度回溯和报告导出
- 全程记录知识来源和证据等级

### 4. 跨平台兼容性

- 纯 Python 实现，无 C++ 依赖
- Windows 友好（自带启动脚本）
- 虚拟环境隔离，依赖管理清晰

## 四、版本信息

- **当前版本**：v6.0（基于 v5.1 稳定版持续迭代）
