# StructPilot v6.0 快速开始

> 3 分钟上手体验

## 第一步：启动应用

### Windows 用户（推荐）

```bash
双击 start.bat
```

### 所有平台手动启动

```bash
# 1. 激活虚拟环境
.venv\Scripts\activate          # Windows
source .venv/bin/activate       # Linux/macOS

# 2. 启动应用
streamlit run main.py --server.port 8501
```

### 浏览器访问

```
http://127.0.0.1:8501/
```

## 第二步：基础配置

### 选择软件体系

侧边栏 → 选择 **cryoSPARC** 或 **RELION**

### 选择回答深度

- **简洁**：快速决策（适合熟手）
- **教学**：详细指导（推荐新手）⭐
- **专家**：深度分析（适合科研决策）

## 第三步：开始第一步

### 方式 1：输入命令

```
在输入框输入：开始 cp_01
```

### 方式 2：点击预设问题

```
点击："如何导入数据？"
```

## 第四步：体验核心功能

### 测试文本问答

```
Q: box size 怎么设置？
Q: CTF 是什么？
Q: 运动校正报错怎么办？
```

### 测试图片上传

```
1. 点击输入框右侧的 📷 图标
2. 上传冷冻电镜相关截图（CTF/2D class/FSC 等）
3. 输入问题："这张图正常吗？"
```

### 测试状态流转

```
依次输入：
- "开始 cp_02"
- "完成 cp_02"
- 查看侧边栏检查点状态变化（绿色 = 已完成）
```

### 测试报告导出

```
1. 切换到"报告"标签页
2. 查看流程总览和对话记录
3. 点击 [导出 Markdown]
```

## 常见问题速查

| 问题 | 解决方法 |
|------|----------|
| 端口被占用 | 换端口：`streamlit run main.py --server.port 8517` |
| Python 版本过低 | 需要 Python 3.10+ |
| 依赖安装失败 | 使用镜像：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple` |
| 浏览器打不开 | 检查防火墙，手动访问 `http://127.0.0.1:8501` |

## AI 增强模式（可选）

如需启用图像理解、语义检索、语音转写：

```bash
# 1. 复制配置模板
cp .env.example .env

# 2. 编辑 .env 文件，填入 API Key
STRUCTPILOT_LLM_API_KEY=your_api_key_here

# 3. 重启应用
```

**不配置也能用！** 基础模式使用本地规则和知识库，无需联网。

## 典型使用场景

### 场景 1：新手学习流程

```
1. 选择"教学"模式
2. 按 cp_01 → cp_02 → ... → cp_12 逐步推进
3. 每步输入"开始"和"完成"
4. 遇到不懂的概念立即提问
5. 最后导出完整学习报告
```

### 场景 2：参数咨询

```
1. 选择"专家"模式
2. 直接提问：
   "cryoSPARC patch motion correction 的 bfactor 参数，
    对于 300kV K3 相机数据，推荐值是多少？"
3. 查看回答中的参数边界和依据
```

### 场景 3：故障排查

```
1. 上传报错截图
2. 输入错误信息
3. 查看诊断信息中的排查路径
4. 按建议调整参数
```

### 场景 4：质控判断

```
1. 上传 CTF/2D class/FSC 截图
2. 问："这张图正常吗？"
3. 系统返回质控标准和改进建议
```

## 进阶功能

### 会话管理

```
侧边栏 → 会话管理
- [+ 新建会话]：开始新项目
- 选择历史会话：恢复之前的进度
- [✏️ 重命名]：给会话命名（如"ApoA1_20260718"）
- [🗑️ 删除]：清理不需要的会话
```

### 知识库沉淀

```
1. 在有价值的回答下点击 [沉淀经验]
2. 编辑标题和内容
3. 选择分类（参数/质控/故障/概念）
4. 保存为草稿，待管理员审核
```

### 诊断信息

```
展开对话中的"诊断信息"查看：
- 问题理解结果
- 关键词提取
- 知识库检索命中
- RAG 相似度得分
- 处理耗时
```
