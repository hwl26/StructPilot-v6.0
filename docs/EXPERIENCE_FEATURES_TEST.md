# 课题组经验功能测试指南

## ✅ 已实现的四个核心功能

### 1. 审核写入机制 ✅
**功能**：经验提交后标记为 `pending`（待审核）状态

**测试步骤**：
1. 切换到「高级模式」
2. 进入「贡献经验」tab
3. 填写标题、症状、解决方案
4. 点击「提交经验」
5. 打开 `knowledge_base/lab_experience_kb.json`
6. 查看最新条目的 `"status": "pending"`

**预期结果**：
- 提交成功提示：「✅ 经验已提交，待管理员审核后生效（目前标注「待验证」，已可检索）」
- JSON 文件中新增条目，status 为 `pending`
- 在经验卡片中显示「⚠️ 待验证」徽章

---

### 2. 搭配报错截图 ✅
**功能**：贡献经验时可上传最多 3 张截图

**测试步骤**：
1. 进入「高级模式」→「贡献经验」
2. 填写基本信息
3. 点击「上传报错截图或结果对比图」
4. 选择 1-3 张图片（支持 png/jpg/jpeg，每张<5MB）
5. 预览上传的图片
6. 提交经验
7. 查看 `runtime/experience_images/` 目录，确认图片已保存
8. 在经验卡片中查看截图

**预期结果**：
- 表单中预览上传的图片
- 提交成功提示：「✅ 经验已提交（含 X 张截图）...」
- `runtime/experience_images/` 目录中有新文件：`cp_XX_YYYYMMDD_HHMMSS_0.png`
- 经验卡片展示「📸 相关截图」区域

---

### 3. 自然语言 + 术语规范化 ✅
**功能**：自动将术语别名转为标准形式，并提取标签

**测试步骤**：
1. 贡献经验时，标题输入：「MC时motion太大怎么办」
2. 症状描述：「运行motioncorr报错，CTF估不准」
3. 不填写标签（让系统自动生成）
4. 提交后查看 JSON 文件

**预期结果**：
- 标题规范化为：「Motion Correction 时 motion 太大怎么办」
- 症状规范化为：「运行 Motion Correction 报错，CTF Estimation 估不准」
- 自动生成标签：["motion", "motioncorr", "ctf", ...]
- 提交成功提示：「... 自动生成 X 个标签 ...」

---

### 4. 类似经验推送（质检失败时）✅
**功能**：质检失败时，基于问题描述智能推送相关经验

**测试步骤**：
1. 切换到「入门模式」
2. 在某个步骤点击「检查结果」
3. 如果质检失败，观察是否显示「🥇 课题组有人遇到过类似问题」
4. 查看推送的经验是否与当前步骤相关

**预期结果**：
- 质检失败时自动触发经验推送
- 推送的经验基于关键词相似度匹配
- 优先显示 `status: approved` 的经验
- 经验卡片展开，包含截图（如果有）

---

## 🧪 完整测试流程

### 场景 A：新手遇到问题 → 推送经验 → 解决问题
1. 入门模式，进行到「CTF Estimation」
2. 点击「检查结果」→ 质检失败（假设分辨率不达标）
3. 系统推送 2 条相关经验：
   - 「CTF 分辨率低的常见原因」（含截图）
   - 「Voltage 参数设置错误」
4. 用户查看经验，按照建议调整参数
5. 重新运行，质检通过

### 场景 B：用户解决问题后贡献经验
1. 切换到高级模式
2. 进入「贡献经验」tab
3. 填写：
   - 标题：「CTF resolution 太低的解决方案」
   - 症状：「CTF结果只有 8Å，预期应该 <4Å」
   - 解决：「检查 Voltage 参数，发现设置成 200kV 但实际是 300kV」
   - 上传 2 张截图：报错界面、参数修改后
4. 提交成功
5. 管理员审核通过（手动修改 JSON 中 status 为 `approved`）
6. 下次其他用户遇到类似问题时，系统自动推送这条经验

## 🔍 手动审核流程（当前版本）

由于完整的管理员审核面板是 P1 优先级（后续实现），当前版本采用手动审核：

1. 打开 `knowledge_base/lab_experience_kb.json`
2. 找到 `"status": "pending"` 的条目
3. 检查内容质量
4. 如果通过，修改为 `"status": "approved"`
5. 如果拒绝，修改为 `"status": "rejected"`
6. 保存文件，刷新应用

**下一版本将实现**：
- 管理员审核面板（Web UI）
- 批量通过/拒绝
- 质量评分和点赞功能
