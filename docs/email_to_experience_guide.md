# 邮件转经验 - 配置指南

## 适用场景

- 不想用 Telegram
- 国内环境，微信/QQ 企业 Bot 门槛太高
- 手机上随时发邮件记录实验经验
- 兼容所有邮箱（Gmail / QQ邮箱 / 163邮箱 / 企业邮箱）

## 配置步骤

### 方案A：使用 QQ 邮箱（推荐国内用户）

#### 第1步：开启 IMAP 服务

1. 登录 https://mail.qq.com/
2. 点击「设置」→「账户」
3. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务」
4. 开启「IMAP/SMTP服务」
5. 点击「生成授权码」（会要求手机验证）
6. **复制这个授权码**（类似：`abcdefghijklmnop`）

#### 第2步：在 StructPilot 中配置

打开 StructPilot → 高级模式 → 设置 → 邮件转经验：

```
IMAP 服务器：imap.qq.com
端口：993
账号：你的QQ邮箱地址（如 123456789@qq.com）
密码：刚才生成的授权码（不是QQ密码！）
启用SSL：✅ 勾选
```

点击「测试连接」，成功后点击「启用邮件监听」

---

### 方案B：使用 Gmail（国际用户）

#### 第1步：开启两步验证并生成应用专用密码

1. 访问 https://myaccount.google.com/security
2. 开启「两步验证」
3. 搜索「应用专用密码」
4. 选择「邮件」→「其他设备」→ 输入「StructPilot」
5. 生成16位密码（如：`abcd efgh ijkl mnop`）
6. **复制这个密码**

#### 第2步：在 StructPilot 中配置

```
IMAP 服务器：imap.gmail.com
端口：993
账号：你的Gmail地址
密码：刚才生成的应用专用密码
启用SSL：✅ 勾选
```

### 方案C：使用 163 邮箱

#### 第1步：开启 IMAP

1. 登录 https://mail.163.com/
2. 设置 → POP3/SMTP/IMAP
3. 开启 IMAP/SMTP 服务
4. 设置客户端授权密码

#### 第2步：配置

```
IMAP 服务器：imap.163.com
端口：993
账号：你的163邮箱
密码：客户端授权密码
启用SSL：✅ 勾选
```

## 使用方法

### 日常记录

**手机上发邮件**：

1. 打开邮件 App（系统自带 / QQ邮箱 / Gmail App 均可）
2. 写新邮件：
   - **收件人**：配置的邮箱地址（发给自己）
   - **主题**：`#经验 Motion Correction 漂移问题解决`
   - **正文**：
     ```
     今天遇到 Motion Correction 后 drift plot 超出范围的问题。
     
     症状：很多 micrograph 被标记为红色，运动校正失败率超过30%
     
     解决方法：
     1. 检查采集时是否有机械振动
     2. 增大 B-factor 参数从 150 → 300
     3. 如果还不行，尝试 Local Motion Correction
     
     最后成功率提升到 95%
     ```
3. 发送

### 自动处理流程

1. StructPilot 每 5 分钟检查一次邮箱
2. 发现主题包含 `#经验` 的邮件
3. 解析内容，创建经验条目（状态：待审核）
4. 在管理员审核面板中显示
5. 审核通过后进入知识库

### 查看记录

- 高级模式 → 设置 → 管理员审核
- 看到来源标注「📧 邮件」的经验
- 点击通过即可

## 常见问题

**Q：密码填什么？**
A：不是邮箱登录密码！是「授权码」或「应用专用密码」，需要在邮箱设置中生成

**Q：提示连接失败？**
A：检查：
1. 是否开启了 IMAP 服务
2. 密码是否填的是授权码（不是登录密码）
3. 端口是否填 993
4. SSL 是否勾选

**Q：多久检查一次邮箱？**
A：每 5 分钟检查一次。如需立即生效，重启 StructPilot

**Q：会删除我的邮件吗？**
A：不会！只是读取邮件内容，原邮件保留在邮箱中

**Q：能同时监听多个邮箱吗？**
A：目前仅支持一个邮箱。如需监听多个，可以设置邮件转发

## 对比：邮件 vs Telegram Bot

| 功能 | 邮件转经验 | Telegram Bot |
|------|-----------|-------------|
| **手机使用** | ✅ 自带邮件 App | ✅ 需安装 Telegram |

| **配置难度** | 🟡 需生成授权码 | 🟢 更简单 |

| **支持图片** | ✅（附件） | ❌ 仅文字 |

**推荐**：
- 国内用户 → 邮件转经验
- 国际用户 / 有科学上网 → Telegram Bot
