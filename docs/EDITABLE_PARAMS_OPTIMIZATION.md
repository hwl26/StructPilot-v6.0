# 参数推荐交互优化说明

**优化时间**：2026-07-22  
**针对问题**：用户反馈"推荐参数应该可编辑，因为收数据后电镜管理员会提供准确参数"

---

## 🎯 用户场景分析

### 真实工作流

```
用户收数据
    ↓
电镜管理员提供准确参数
    ↓
   例如：
   · 像素尺寸：0.826 Å/pix（而非系统推荐的 0.86）
   · 加速电压：300 kV
   · 总剂量：50 e-/Å²
    ↓
用户需要将这些准确值填入系统
    ↓
开始数据处理流程
```

**核心矛盾**：
- 系统推荐值是"经验估算"（如 Krios 300kV → pixel size 0.86）
- 用户实际值是"准确测量"（如 0.826 Å/pix）
- **像素尺寸错误 1%，可能导致 3D 重构失败！**

## ❌ 原版设计问题

### 界面截图（你的反馈）

```
推荐参数：
加速电压：300 kV          像素尺寸：0.86 Å/pix
颗粒直径：150 Å           掩膜直径：180 Å
CTF 最大分辨率：5 Å       2D 分类数：50
```

**问题**：
1. ❌ **只读展示**：无法修改
2. ❌ **无区分度**：所有参数看起来一样重要
3. ❌ **缺少警示**：不知道哪些参数必须准确

## ✅ 新版设计：分级可编辑

### 设计原则

| 原则 | 说明 | 实现 |
|------|------|------|
| **关键参数突出** | 像素尺寸/电压错误 → 全流程失败 | 🔴 红色标记 + 必填 * |
| **推荐值作默认** | 系统推荐值预填，用户可修改 | 输入框 value=推荐值 |
| **次要参数折叠** | 颗粒直径等可后续调整 | 🟡 黄色折叠区 |
| **提供退路** | 误改后可恢复推荐值 | "🔄 恢复推荐值"按钮 |

### 关键改进点

#### 1. 分级展示（红色 vs 黄色）

| 级别 | 参数 | 重要性 | 可编辑时机 |
|------|------|--------|----------|
| 🔴 关键 | 加速电压、像素尺寸 | 错误 → 全流程失败 | 必须在开始前填准确 |
| 🟡 推荐 | 颗粒直径、掩膜直径、CTF分辨率、2D分类数 | 错误 → 部分步骤失败 | 可在流程中调整 |

#### 2. 输入框 + 推荐值提示

```python
voltage = st.number_input(
    "加速电压 (kV) *",
    value=300,  # 推荐值作为默认值
    min_value=100,
    max_value=500,
    step=100,
    help="必须准确！通常为 200kV 或 300kV",
)
```

**用户体验**：
- 看到推荐值 300
- 可以直接修改为 200（如果是 Arctica）
- help 提示说明为什么重要

#### 3. 警告文字（关键参数）

```
⚠️ 此参数错误会导致全流程失败！请使用电镜管理员提供的准确值
```

**目的**：
- 强调像素尺寸的关键性
- 引导用户找电镜管理员确认

#### 4. 恢复推荐值按钮

**场景**：用户误改后不记得原推荐值

**实现**：
```python
if st.button("🔄 恢复推荐值"):
    # 重新生成推荐，覆盖编辑
    fresh_workflow = _generate_workflow_recommendation(profile)
    st.session_state._edited_params = fresh_workflow["params"].copy()
    st.rerun()
```

## 🔧 技术实现

### 状态管理

```python
# 临时存储用户编辑的参数
if "_edited_params" not in st.session_state:
    st.session_state._edited_params = params.copy()

# 每次输入框变化时更新
voltage = st.number_input(..., key="param_voltage")
st.session_state._edited_params["voltage"] = voltage

# 确认时保存到 workflow
workflow["params"] = st.session_state._edited_params.copy()
st.session_state.recommended_workflow = workflow
```

## 📝 用户指南（需补充到文档）

### 如何使用推荐参数

1. **查看推荐值**：系统已根据你的设备和样品预填
2. **修正关键参数**：
   - 找电镜管理员确认"像素尺寸"和"加速电压"
   - 将准确值填入输入框
3. **调整推荐参数**（可选）：
   - 点击"🟡 推荐参数"展开
   - 根据经验调整颗粒直径、2D分类数等
4. **确认并开始**：检查无误后点击"确认并开始"

### 常见问题

**Q1: 不知道像素尺寸怎么办？**
A: 找电镜管理员确认，或查看采集时的记录文件（通常在 .mdoc 或 log 文件中）

**Q2: 改错了怎么办？**
A: 点击"🔄 恢复推荐值"按钮，恢复系统推荐的默认值

**Q3: 推荐参数可以后续修改吗？**
A: 黄色折叠区的参数（颗粒直径等）可以在流程中随时调整
