# 动态Tab优化说明

**优化时间**：2026-07-22  
**针对问题**：教学/入门模式下"报告导出"Tab为空白

---

## 🎯 问题诊断

### 用户反馈

**截图显示**：
- 当前模式：教学模式
- 顶部Tab：对话陪跑 | 报告导出 | 设置
- 点击"报告导出" → 内容区域空白

**问题**：
- ❌ 报告导出功能只在高级模式下有用（参数导出/会话保存）
- ❌ 入门/教学模式用户点击后看到空白，困惑
- ❌ 增加认知负担："为什么有这个Tab但没内容？"

---

## ✅ 优化方案

### 方案对比

| 方案 | 描述 | 优点 | 缺点 |
|------|------|------|------|
| **方案1** | 按模式动态显示Tab | 简洁，不显示无用Tab | 需修改Tab逻辑 |
| **方案2** | 空白Tab显示引导 | 引导用户了解高级功能 | 仍需一次点击 |

**选择**：**方案1 - 按模式动态显示Tab**

---

## 🛠️ 实现逻辑

### Tab显示规则

```python
if app_mode in ["beginner", "teaching"]:
    显示Tab：对话陪跑 | 设置
else:  # expert / 原v6
    显示Tab：对话陪跑 | 报告导出 | 设置
```

### 代码实现

```python
# main.py 3729行

# 动态Tab：根据模式决定显示哪些Tab
_app_mode = st.session_state.get("app_mode", "beginner")
if _app_mode in ["beginner", "teaching"]:
    # 入门/教学模式：只显示对话和设置
    tab_labels = ["对话陪跑", "设置"]
    tab_chat, tab_settings = st.tabs(tab_labels)
    tab_report = None  # 不显示报告导出
else:
    # 高级模式/原v6：显示全部Tab
    tab_labels = ["对话陪跑", "报告导出", "设置"]
    tab_chat, tab_report, tab_settings = st.tabs(tab_labels)

# 报告导出Tab（仅高级模式）
if tab_report is not None:
    with tab_report:
        st.markdown("### 流程报告")
        ...
```

## 🎨 为什么这样设计

### 设计原则

**"按需显示"原则**：
> 只显示用户当前模式下有用的功能，隐藏无用或空白的选项。

**理由**：
1. **减少选择疲劳**：Tab越多，用户越不知道点哪个
2. **避免挫败感**：点到空白Tab会让用户觉得"这软件不完整"
3. **引导路径清晰**：入门用户只需关注"对话陪跑"，不需要"报告导出"

---

### 为什么保留"设置"Tab

**所有模式都保留"设置"的原因**：

1. **LLM配置**：入门用户也可能需要切换LLM（如从OpenAI切到本地Ollama）
2. **桌宠设置**：所有用户都可能调整桌宠大小/状态
3. **通用性**：设置Tab内容对所有用户都有意义

---

### 为什么隐藏"报告导出"

**入门/教学模式不需要"报告导出"的原因**：

1. **功能定位**：
   - 入门模式：快速完成任务，不需要导出报告
   - 教学模式：学习原理，不需要导出参数

2. **报告导出的用途**：
   - 导出参数为CSV/JSON（批量处理用）
   - 保存会话历史（复盘用）
   - 生成Markdown报告（汇报用）
   - **这些都是高级用户的需求**

3. **替代方案**：
   - 入门用户想要报告 → 引导他切换到高级模式
   - 教学用户想要记录 → 引导他完成学习后切换

## 🔧 技术实现细节

### 关键改动

**文件**：`main.py`

**位置**：3729-3740行（Tab定义）、4615-4631行（报告导出内容）

**改动1：动态Tab定义**
```python
# 原版（固定3个Tab）
tab_chat, tab_report, tab_settings = st.tabs([
    "对话陪跑", "报告导出", "设置"
])

# 新版（动态2-3个Tab）
_app_mode = st.session_state.get("app_mode", "beginner")
if _app_mode in ["beginner", "teaching"]:
    tab_chat, tab_settings = st.tabs(["对话陪跑", "设置"])
    tab_report = None
else:
    tab_chat, tab_report, tab_settings = st.tabs([
        "对话陪跑", "报告导出", "设置"
    ])
```

**改动2：条件渲染报告导出**
```python
# 原版（总是渲染）
with tab_report:
    st.markdown("### 流程报告")
    ...

# 新版（条件渲染）
if tab_report is not None:
    with tab_report:
        st.markdown("### 流程报告")
        ...
```
