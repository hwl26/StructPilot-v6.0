### 2. 质控 tab 优化 ✅
- 删除冗余的"课题组经验"区域

- 补充"常见陷阱"说明

### 3. SOP tab 优化 ✅
- 操作步骤图文排版
- 从 CSV 读取实验室参数建议
- 红框高亮展示：实验室值 vs 官方值
- 显示参数差异和调整建议

### 4. 参数区优化 ✅
- 参数推荐结合官方文档+平台实际情况
- 官方补充说明功能模块
- 推荐修改参数（实验室经验+官方对比）
- 课题组 workflow 参数展示

### 5. 截图区优化 ✅
- 图文排版支持
- 无图时提供完整上传指引
- 显示本阶段关键观察点

### 6. 对话上下文优化 ✅
- 增强上下文连贯性：10轮对话历史（原6轮）
- 每条消息保留 200 字符（原100字符）
- 智能沉淀经验：问答对提取+上下文信息
- 会话摘要长度：1500 字符（原1200字符）

## 🚀 快速启动

### Windows 用户

```bash
# 方法1：使用启动脚本（推荐）
start.bat

# 方法2：手动启动
pip install -r requirements.txt
streamlit run main.py
```

### macOS/Linux 用户

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行应用
streamlit run main.py
```

### 首次启动配置

1. 应用会自动在浏览器打开（默认 http://localhost:8501）
2. 左侧边栏选择软件（cryoSPARC 或 RELION）
3. 可选：配置 LLM API（设置 → LLM 配置）
4. 开始使用 14 步流程导航

## 🔬 创新点

### 1. 智能上下文管理
- **10 轮对话历史**（200字符/条），确保多轮对话连贯性
- **会话摘要自动更新**，包含近期讨论主题
- **1500 字符摘要长度**，保留更多关键信息

### 2. 实验室参数对比系统
- 从 `lab_parameters_master.csv` 读取实验室实际使用参数
- **红框高亮卡片**展示：实验室值 vs 官方值
- 显示**参数差异**和**调整建议**
- 支持 60+ 参数的详细对比

### 3. 智能沉淀经验
- 自动提取**最新问答对**（用户问题+助手回答）
- 附加**上下文信息**（checkpoint、software、params）
- 智能匹配**用户问题**（向前查找最近5条消息）
- 确保知识条目的**完整性和可理解性**

### 4. 图文排版增强
- 有图时：图片 + 标题 + 说明
- 无图时：上传指引 + 关键观察点 + 使用说明
- 统一的视觉风格和交互体验

### 5. 双软件官方文档集成
- RELION 官方文档自动匹配当前检查点
- cryoSPARC 官方文档占位（可扩展）
- 官方文档链接点击跳转
