**最推荐！专为 Streamlit 设计**

- ✅ 5 分钟部署
- ✅ 自动 HTTPS

### 限制

## 方案 2：Hugging Face Spaces ⭐⭐⭐⭐

**AI/ML 应用的理想平台**

### 优点
- ✅ 完全免费
- ✅ 2GB RAM（比 Streamlit Cloud 大）
- ✅ GPU 支持（付费）
- ✅ 社区友好

### 限制
- 必须公开应用
- 适合 AI/ML 应用

### 快速部署

1. **注册账号**
   访问 [huggingface.co](https://huggingface.co/join)

2. **创建 Space**
   - 点击 "New Space"
   - 选择 "Streamlit" 作为 SDK
   - 命名：`structpilot`

3. **上传代码**

```bash
# 克隆 Space 仓库
git clone https://huggingface.co/spaces/你的用户名/structpilot
cd structpilot

# 复制代码
cp -r StructPilot_v6/* .

# 创建 README.md（必需）
cat > README.md << 'EOF'
---
title: StructPilot
emoji: 🔬
colorFrom: blue
colorTo: green
sdk: streamlit
sdk_version: 1.35.0
app_file: main.py
pinned: false
---

# StructPilot

冷冻电镜数据处理智能陪跑助手
EOF

# 提交并推送
git add .
git commit -m "Initial commit"
git push
```

4. **配置环境变量**
   在 Space 设置中添加：
   - `STRUCTPILOT_LLM_API_KEY`
   - `STRUCTPILOT_EMBEDDING_API_KEY`
   - 等等

5. **访问应用**
   URL：`https://huggingface.co/spaces/你的用户名/structpilot`

## 方案 3：Railway ⭐⭐⭐⭐

**现代化部署平台**

### 优点
- ✅ $5 免费额度/月
- ✅ 支持 Docker
- ✅ 自动 HTTPS
- ✅ 数据库支持

### 限制
- 每月 $5 额度用完后停止
- 需要绑定信用卡（不扣费）

### 快速部署

1. **注册账号**
   访问 [railway.app](https://railway.app)，使用 GitHub 登录

2. **新建项目**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 选择你的 StructPilot 仓库

3. **配置环境变量**
   在 Variables 中添加：
   ```
   STRUCTPILOT_LLM_API_KEY=你的密钥
   PORT=8501
   ```

4. **添加启动命令**
   在 Settings → Deploy 中设置：
   ```bash
   streamlit run main.py --server.port=$PORT --server.address=0.0.0.0
   ```

5. **生成域名**
   在 Settings → Networking 中点击 "Generate Domain"

## 方案 4：Render ⭐⭐⭐

**简单的云部署平台**

### 优点
- ✅ 完全免费版
- ✅ 自动 HTTPS
- ✅ 支持 Docker

### 限制
- 免费版 512MB RAM
- 不活跃会睡眠
- 唤醒较慢（30-60 秒）

### 快速部署

1. **注册账号**
   访问 [render.com](https://render.com)

2. **创建 Web Service**
   - 点击 "New +"
   - 选择 "Web Service"
   - 连接 GitHub 仓库

3. **配置服务**
   - **Name**: `structpilot`
   - **Region**: `Oregon` 或其他
   - **Branch**: `main`
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `streamlit run main.py --server.port=$PORT --server.address=0.0.0.0`

4. **添加环境变量**
   在 Environment 中添加 API Keys

5. **部署**
   点击 "Create Web Service"

## 方案 5：Fly.io ⭐⭐⭐

**全球分布式部署**

### 优点
- ✅ 3 个免费实例
- ✅ 全球 CDN
- ✅ 支持 Docker

### 限制
- 需要信用卡验证
- 配置较复杂

### 快速部署

1. **安装 CLI**
   ```bash
   # Windows
   powershell -Command "iwr https://fly.io/install.ps1 -useb | iex"
   
   # Linux/Mac
   curl -L https://fly.io/install.sh | sh
   ```

2. **登录**
   ```bash
   flyctl auth login
   ```

3. **初始化应用**
   ```bash
   cd StructPilot_v6
   flyctl launch
   ```

4. **部署**
   ```bash
   flyctl deploy
   ```

## 方案 6：Oracle Cloud（永久免费）⭐⭐⭐⭐

**最强免费额度**

### 优点
- ✅ 永久免费（真正的免费服务器）
- ✅ 2 个免费虚拟机（ARM）
- ✅ 4 核 24GB RAM（合计）
- ✅ 200GB 存储
- ✅ 10TB 月流量

### 限制
- 注册较复杂（需要信用卡验证）
- 需要自己配置服务器

### 快速部署

1. **注册账号**
   访问 [oracle.com/cloud/free](https://www.oracle.com/cloud/free/)

2. **创建实例**
   - 选择 "Compute → Instances"
   - 创建 VM.Standard.A1.Flex（ARM）
   - 4 OCPU, 24GB RAM（免费额度内）
   - Ubuntu 22.04

3. **配置安全规则**
   开放端口：22, 80, 443

4. **部署应用**
   按照 [03_云服务器部署.md](03_云服务器部署.md) 步骤操作
